# DevelopersHub Internship - Data Analytics/Data Science Projects

This repository contains all projects completed during my Data Analytics/Data Science internship at DevelopersHub Corporation.

## Projects

### Task 1 — Iris Dataset EDA
* Exploratory Data Analysis on Iris dataset
* Visualized feature distributions and relationships
* Tools: Python, Pandas, Matplotlib, Seaborn

### Task 2 — Credit Risk Prediction
* Predicted loan default using classification models
* Logistic Regression: ~79% accuracy, Decision Tree: ~69%
* Tools: Python, Pandas, Scikit-learn

### Task 3 — Customer Churn Prediction
* Predicted bank customer churn using Random Forest
* Achieved 87.1% accuracy
* Analyzed feature importance
* Tools: Python, Pandas, Scikit-learn

### Task 4 — Insurance Claim Prediction
* Predicted medical insurance charges using Linear Regression
* Visualized impact of Age, BMI, and Smoking status
* Evaluated using MAE and RMSE
* Tools: Python, Pandas, Scikit-learn, Matplotlib

### Task 5 — Personal Loan Acceptance Prediction
* Predicted which customers are likely to accept a personal loan offer
* Logistic Regression & Decision Tree classifier trained on Bank Marketing Dataset
* Analyzed customer groups by age, job, education, and balance
* Key Insight: Students, Retired persons, and Tertiary educated customers accept loans most
* Tools: Python, Pandas, Scikit-learn, Matplotlib, Seaborn
